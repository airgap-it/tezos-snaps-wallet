How To Connect

The Beacon Debug Wallet can be used to simulate a connection between a wallet and a dApp. All relevant details are exposed and in the Beacon Debug Wallet to simplify testing and debugging.

With the Beacon Debug Wallet dApp developer can also view their application from the perspective of specific addresses.

How To Connect A dApp To The Beacon Debug Wallet
  1. Go to the dApp you want to establish a connection.
  2. Use the functionality on the dApp to open the Beacon dialog. Usually this is a button labelled "Connect Wallet" or "sync".
  3. Select "Pair with another device" on the beacon dialog.
  4. Hover over the QR code, click on the Copy button, the Beacon Connection code has now been copied to your clipboard.
  5. Come Back to the Beacon Debug Wallet and click on the button "Connect To A New dApp"
